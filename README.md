# careercompass
A student career guidance platform that calculates CGPA, predicts suitable career paths, recommends skills to learn, provides interview preparation resources, and tracks placement readiness

## Overview
CareerCompass is a student career guidance platform that helps students plan their academic and professional journey.

## Features
- CGPA Calculator
- Career Prediction
- Skill Gap Analysis
- Learning Roadmap
- Interview Preparation
- Placement Readiness Assessment

## Technology Stack
- Python
- Machine Learning
- Data Analytics
- Streamlit
- Pandas

## Future Enhancements
- AI-powered recommendations
- Resume analysis
- Personalized learning plans
- Placement prediction dashboard
